# Changelog

Все значимые изменения в проекте фиксируются в этом файле.

## [1.1.0] - 2026-07-22

### Добавлено
- Шапка меню теперь показывает актуальную статистику: количество игр в библиотеке и суммарное наигранное время (`[ ИГР: N ] [ ВСЕГО ИГРАНО: XhYm ]`).
- Обработка неверного ввода: при выборе несуществующего номера игры выводится явное сообщение вместо молчаливого возврата в меню.
- При добавлении игры (`A`) выводится сообщение, если диалог выбора `.exe`-файла был закрыт без выбора.
- При старте скрипт дополнительно закрывает Steam, Epic Games Launcher, Discord, Telegram, WhatsApp (не только `explorer.exe`).
- В меню удаления игры (`D`) добавлен пункт `[0] Назад в главное меню`.

### Исправлено
- Удаление игры из библиотеки (`D`) теперь корректно чистит и `Статистика.txt` — раньше запись об удалённой игре оставалась там навсегда.
- Временные файлы (`%temp%\*.tmp`) теперь удаляются перед созданием новых, что исключает использование "прилипших" файлов от предыдущего некорректного завершения скрипта.

### Изменено
- Убрана статичная верхняя панель `[ USER_01 ] [ время запуска ] [ PSN ] [ FW ]` — время в ней не обновлялось и вводило в заблуждение. Заменена на панель с реальной статистикой библиотеки.

## [1.0.0]
- Первая версия батника: библиотека игр, добавление/удаление через текстовое меню, учёт времени игры, автоматическая выгрузка фонового софта и очистка `Temp` при запуске игры.
