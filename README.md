# 🎮 Игровой лаунчер-оптимизатор для слабых ПК

Удобный интерактивный скрипт (`Games.bat`), который временно разгружает операционную систему, выгружая всё лишнее из оперативной памяти для повышения стабильности и FPS в играх.

> 🤖 *Этот код, а также данное описание созданы с помощью нейросети Gemini по задумке и под руководством автора проекта!*

---

## 💡 О проекте: Для каких игр это создано?

В первую очередь этот скрипт и вся сборка ориентированы на **одиночные игры**. Это практичный способ освободить ресурсы на тяжелых версиях Windows 11.

### ➕ Плюсы использования:
* **Освобождение ресурсов:** Отключение графической оболочки Windows освобождает процессор и ОЗУ для запущенной игры.
* **Чистота системы:** В фоне не будут запускаться случайные задачи, обновления или уведомления, которые могут вызывать фризы.
* **Удобная игротека:** Новые игры можно добавлять прямо через интерфейс лаунчера с помощью графического окна выбора файлов.
* **Безопасность:** Скрипт ничего не удаляет безвозвратно. При выходе система полностью возвращается в стандартный рабочий вид.

### ➖ Минусы и жесткие ограничения:
* **Не используйте сторонние лаунчеры:** Батник создан **специально для пиратских версий игр** (или игр без DRM, запускаемых напрямую через собственный `.exe`)! Даже не пытайтесь запускать через него игры из Steam, Epic Games Launcher, EA App и других официальных клиентов. Они автоматически подтянут свои тяжелые фоновые сервисы, что полностью уничтожит весь смысл оптимизации. 
* **Скачивайте игры только с проверенных сервисов**, чтобы не занести в систему вирусы, так как во время игры фоновая защита будет ограничена.
* **Временное ограничение функций:** Пока вы находитесь в игре, у вас не будет доступа к рабочему столу, браузеру и мессенджерам (они принудительно закрываются ради максимальной экономии памяти).

## 🛠️ Как пользоваться

1. **Скачайте** файлы `Games.bat` и `Игры.txt` и положите их в одну папку.
2. **Запустите** file `Games.bat` (права администратора не требуются).
3. Рабочий стол скроется, и откроется текстовое меню терминала.
4. Выберите игру по цифре или нажмите `A`, чтобы добавить новую.
5. При выборе пункта `[0] ВЫХОД` ваш рабочий стол и все функции Windows полностью восстановятся.

### 💡 Совет для удобства и красоты:
Нажмите правой кнопкой мыши по файлу `Games.bat` ➔ *Отправить* ➔ *Создать ярлык на рабочем столе*. 
В свойствах ярлыка нажмите *«Сменить значок»* и выберите любую красивую иконку (например, в виде геймпада). Теперь запускать лаунчер можно прямо с рабочего стола.

### ⚙️ Настройка стартовой папки для выбора игр:
По умолчанию окно добавления игр открывается на диске `C:\`. Если ваши игры хранятся, например, в `D:\Games`, можно сделать так, чтобы окно сразу открывалось там:
1. Откройте файл `games.bat` через Блокнот.
2. Нажмите `Ctrl + F` и найдите строку: `selected_path.tmp`
3. Чуть правее в этой строке найдите параметр: `$d.InitialDirectory='%SystemDrive%\';`
4. Замените его на свой путь, например: `$d.InitialDirectory='D:\Games\';`
5. Сохраните файл (`Ctrl + S`).

📌 **Что делать, если сбойнул рабочий стол?**
Если у вас выключили свет, зависла игра или вы случайно закрыли окно батника на крестик, из-за чего рабочий стол не вернулся сам:
1. Нажмите `Ctrl + Alt + Del` и выберите **«Диспетчер задач»**.
2. Нажмите *«Запустить новую задачу»* (или *Файл* ➔ *Запустить новую задачу*).
3. Введите `explorer.exe` и нажмите Enter. Рабочий стол вернется!

---

## 🧠 Что конкретно делает батник?

1. **Временно останавливает Проводник Windows (`explorer.exe`):** Освобождает ОЗУ и разгружает процессор.
2. **Закрывает фоновые программы:** Автоматически завершает работу Discord, Telegram, WhatsApp, Steam и Epic Games на время сессии, чтобы они не расходовали память в фоне.
3. **Закрывает Google Chrome:** Принудительно завершает процессы браузера.
4. **Чистит мусор:** Очищает временные папки системы (`Temp`), не затрагивая кэш шейдеров.
5. **Восстановление системы:** При закрытии игрового сеанса автоматически перезапускает оболочку Windows.

Этот батник — часть моей личной комплексной настройки для выжимания максимума из слабого железа. 

**Мои результаты с запущенным батником:**
* Потребление видеопамяти (VRAM): упало до **0.1 ГБ**!
* Потребление оперативной памяти (ОЗУ): всего **2.3 ГБ** (иногда меньше).
* Количество процессов в системе: снизилось до **75–80 штук**.

**Что я сделал со своей Windows для достижения таких результатов:**
1. **ОС:** Использую Windows 11 Корпоративная (Enterprise).
2. **Защита и Браузер:** Вырезал встроенный антивирус (Защитник Windows) и браузер Microsoft Edge через сторонние утилиты-деблоатеры.
3. **Твики системы:** С помощью утилиты *Chris Titus Tech Windows Utility (WinUtil)* отключил телеметрию, «оптимизацию во весь экран» и другие фоновые службы.
4. **Визуализация:** Отключил абсолютно все анимации Windows (сглаживание шрифтов при этом оставил дефолтным).
5. **Драйверы:** Устанавливал драйвер видеокарты через *NVCleaner*, которая вырезает из инсталлятора телеметрию и лишние сервисы. В системе нет тяжелого "Nvidia App", в фоне ничего не висит.

## 🛠️ Дополнительные утилиты для оптимизации

Для улучшения плавности игрового процесса можно использовать вместе с батником следующие программы:

### 1. Настройка ISLC (Рекомендуется для 16 ГБ ОЗУ)
Программа *Intelligent Standby List Cleaner* очищает кэш памяти и снижает задержки, улучшая взаимодействие железа.

**Параметры для настройки:**
* *The list size is at least* (Размер списка минимум): **1024 MB**
* *Free memory is less than* (Свободная память меньше): **2048 MB**
* Поставьте галочку: *«Enable custom timer resolution»* (Включить пользовательское разрешение таймера).
* *Wanted timer resolution* (Желаемое разрешение таймера): **0.5000**
* *Частота опроса ISLC (мс):* **1000**
* Нажмите кнопку **«Start»**.

### 2. Настройка Special K (Для улучшения плавности и инпут-лага)
Утилита *Special K* помогает настроить правильный фреймпейсинг (время кадра) и обеспечивает хороший инпут-лаг. Вы можете выставить **любую частоту кадров на ваш вкус**, но утилита особенно полезна, если игра не дотягивает до стабильных 60 FPS и вам нужна высокая плавность на более низкой частоте кадра.

* **Шаг 1: Скачивание.** Рекомендуется использовать проверенную версию **v25.3.25** в виде ZIP-архива.
* **Шаг 2: Локальная установка в игру (Внутренняя инъекция).**
  Крайне рекомендуется использовать именно локальную (внутреннюю) инъекцию, так как глобальный режим потребляет гораздо больше системных ресурсов.
  1. Скопируйте файл `SpecialK64.dll` из папки утилиты.
  2. Вставьте его в корневую директорию игры, туда же, где лежит главный исполняемый (`.exe`) файл. 
  3. **Важно по переименованию:** Название `dxgi.dll` используется **строго для игр на DirectX 11, DirectX 12 и Vulkan**. Если вы запускаете игру на более старых версиях графического API (например, DirectX 9 или 10), посмотрите в интернете, как правильно переименовать файл для вашей версии, либо в крайнем случае используйте глобальную инъекцию (но помните, что она более прожорлива к ресурсам).
* **Шаг 3: Настройка внутри игры.**
  Запустите игру через батник. Нажмите `Ctrl + Shift + Backspace` (или кнопку *Home* на геймпаде) для открытия меню Special K.
  1. **Вертикальная синхронизация (Вкладка Display):** Вы можете настроить ограничение кадров (`Framerate Limit`) под ваши задачи. Например, при локе на половинную частоту герцовки монитора (пункт *Half V-Sync*), игра получит ровный график времени кадра без резких скачков.
  2. **Процессор (Вкладка Advanced):** Включите использование спящих ядер (актуально для многопоточных CPU) и поменяйте приоритет процесса игры на *«High»* (Высокий).
  3. **Задержка ввода (Альтернатива):** Функция `Latent Sync` позволяет снизить задержку управления, но ваша видеокарта должна иметь небольшой запас мощности (загрузка не более 70%), иначе на экране может появиться тиринг (разрывы кадра).
  4. **Для видеокарт Nvidia (Вкладка Advanced):** Включите функцию *Nvidia Reflex* в режиме `Reflex + Boost` внутри утилиты (в настройках самой игры Reflex в этом случае лучше выключить). Если появятся нестабильности — просто отключите её.
