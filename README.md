===================================================================
             PC GAME CONSOLE MODE (Оптимизатор для слабых ПК)
===================================================================

Удобный интерактивный скрипт (Games.bat) для геймеров со слабыми ПК.
Он временно "превращает" ваш компьютер в игровую консоль, выгружая 
всё лишнее из оперативной памяти для максимального FPS в играх.

🤖 Этот код, а также данный текст полностью созданы с помощью нейросети 
Gemini по задумке и под руководством автора проекта!

-------------------------------------------------------------------
🎮 ДЛЯ КАКИХ ИГР ЭТОТ БАТНИК?
-------------------------------------------------------------------
В первую очередь этот скрипт и вся сборка созданы для ОДИНОЧНЫХ игр! 
Особенно если вы играете на геймпаде в 30 кадров в секунду. Это лучший 
способ выжать максимум плавности из тяжелой Windows 11.

-------------------------------------------------------------------
📊 МОИ РЕЗУЛЬТАТЫ С ЗАПУЩЕННЫМ БАТНИКОМ:
-------------------------------------------------------------------
* Потребление видеопамяти (VRAM): упало до 0.1 ГБ!
* Потребление оперативной памяти (ОЗУ): всего 2.3 ГБ (иногда даже меньше).
* Количество процессов в системе: снизилось до 75–80 штук.

⚙️ МОЙ ЛИЧНЫЙ СПОСОБ ПОЛНОЙ ОПТИМИЗАЦИИ СИСТЕМЫ:
-------------------------------------------------------------------
Этот батник — часть моей личной сборки для выжимания максимума из железа. 
Вот что я сделал со своей Windows для достижения таких результатов:

1. ОС: Использую Windows 11 Корпоративная (Enterprise).
2. Защита и Браузер: Полностью вырезал встроенный антивирус (Защитник 
   Windows) и браузер Microsoft Edge. Делал это через сторонние утилиты 
   деблоатеры (их исходники легко найти тут же, на GitHub).

   ⚠️ ВНИМАНИЕ: Это исключительно МОЙ ЛИЧНЫЙ ОПЫТ, и я ни в коем случае 
   НЕ ПРИНУЖДАЮ вас его повторять! Удаление Защитника Windows — это 
   серьезный шаг. Если вы решите сделать так же ради экономии ресурсов, 
   помните: вы остаетесь без автоматической защиты. Скачивайте игры и репаки 
   ТОЛЬКО из ультра-проверенных и доверенных источников. Если поймаете 
   скрытый майнер, система начнет тормозить в три раза сильнее, и никакой 
   батник вам уже не поможет. Не уверены — лучше оставьте Защитник включенным!

3. Твики системы: Использовал знаменитую утилиту Chris Titus Tech 
   Windows Utility (WinUtil). С её помощью полностью вырезал телеметрию, 
   отключил "оптимизацию во весь экран" (которая часто ломает FPS) и другие 
   фоновые службы, пожирающие ресурсы.
4. Визуализация: Отключил абсолютно все анимации Windows, но при этом 
   НЕ упрощал текст (сглаживание шрифтов осталось дефолтным и красивым).
5. Драйверы: Устанавливал драйвер видеокарты через специальную утилиту 
   (например, NVCleaner), которая вырезает из инсталлятора весь мусор 
   и телеметрию. В системе нет никакого тяжелого "Nvidia App", в фоне 
   вообще ничего лишнего не висит.

-------------------------------------------------------------------
🧠 НАСТРОЙКА ISLC (Крайне рекомендую для 16 ГБ ОЗУ)
-------------------------------------------------------------------
Если у вас 16 ГБ оперативной памяти, обязательно скачайте утилиту ISLC 
(Intelligent Standby List Cleaner). Она очищает кэш памяти и снижает 
задержки (инпут-лаг), улучшая взаимодействие процессора, видеокарты 
и всего железа.

Выставите в ней следующие идеальные значения:
* Размер списка минимум (The list size is at least): 1024 MB
* Свободная память меньше (Free memory is less than): 2048 MB
* Поставьте галочку: "Включить пользовательское разрешение таймера" 
  (Enable custom timer resolution).
* Желаемое разрешение таймера (Wanted timer resolution): 0.5000
* Частота опроса ISLC (мс): 1000
После этого нажмите кнопку "Начать" (Start).

🛠️ ПОЛНЫЙ ГАЙД ПО НАСТРОЙКЕ SPECIAL К (Для идеальных 30 FPS)
-------------------------------------------------------------------
Если ваш монитор не поддерживает G-Sync или FreeSync, а поиграть хочется 
с идеальной консольной плавностью в 30 кадров и без задержек управления 
(инпут-лага), я крайне рекомендую использовать утилиту Special K. Правильная 
настройка выдает фреймпейсинг и инпут-лаг даже КРУЧЕ, чем на консолях (например, PS4).

Шаг 1: Скачивание
Не рекомендую скачивать самую последнюю версию с официального сайта или 
PlayGround — в некоторых играх она работает нестабильно. 
Самая надежная, свежая и стабильная версия — v25.3.25. Скачивайте её 
желательно в виде ZIP-архива (хотя инсталлер тоже подойдет, не критично).

Шаг 2: Установка в игру (Локальная инъекция)
Это самый стабильный способ, который не ломает систему:
1. Распакуйте архив со Special K и сохраните эту папку где-нибудь под рукой, 
   чтобы иметь к ней быстрый доступ.
2. Найдите в папке утилиты файл "SpecialK64.dll" и скопируйте его.
3. Вставьте этот файл в корневую директорию игры, ТУДА ЖЕ, ГДЕ ЛЕЖИТ главный 
   исполняемый (.exe) файл игры.
   *Внимание:* В некоторых играх .exe файл спрятан глубже основной папки 
   (например, в подпактах "bin", "bin\x64" или "win64"). Ищите именно папку с .exe!
4. Переименуйте скопированный файл "SpecialK64.dll" в "dxgi.dll" (без кавычек).

Шаг 3: Настройка внутри игры
Запустите игру через наш батник. Внутри игры нажмите комбинацию клавиш 
Ctrl + Shift + Backspace (или клавишу Home на геймпаде) для открытия меню Special K.

Выставите следующие настройки:
1. Вертикальная синхронизация (Вкладка Display):
   * Перейдите в верхнюю вкладку "Display".
   * Включите половинный всинк ("Half V-Sync"). 
   * Примечание: Если пункт "Half V-Sync" не появляется сразу, укажите в этой же 
     вкладке точную герцовку вашего монитора (например, 60 Hz), и выбор появится.
   * Также на главном экране утилиты обязательно включите ограничитель кадров 
     (Framerate Limit) строго на значение 30. При мониторе 60 Гц это залочит игру 
     на железные, супер-стабильные 30 FPS с идеальным графиком времени кадра.
2. Процессор (Вкладка Advanced на главном меню):
   * Включите использование спящих ядер (актуально для многопоточных процессоров).
   * Поменяйте приоритет процесса игры на "Высокий" (High).
3. Задержка ввода (Альтернативный способ):
   * Вместо Half V-Sync можно использовать функцию "Latent Sync". Она дает нулевую 
     задержку управления, как вообще без всинка. 
   * ВАЖНОЕ ТРЕБОВАНИЕ: Ваша видеокарта должна иметь запас мощности и не должна быть 
     загружена более чем на 70%. Иначе по экрану пойдут горизонтальные полосы 
     разрыва кадра (тиринг). Если запас по мощности есть — включайте не задумываясь.
4. Для видеокарт Nvidia (Вкладка Advanced):
   * Включите функцию Nvidia Reflex.
   * Рекомендуется выставить режим "Reflex + Boost" именно ВНУТРИ утилиты Special K, 
     а в настройках самой игры Reflex лучше выключить. 
   * Важно: Если после этого в игре появились странные лаги, которых раньше не было, 
     просто отключите Reflex в утилите.

-------------------------------------------------------------------
1. Убивает Проводник Windows (explorer.exe): Освобождает ОЗУ и процессор[cite: 1].
2. Закрывает Google Chrome: Принудительно завершает прожорливый браузер[cite: 1].
3. Чистит мусор: Очищает временные папки (Temp), не трогая кэш шейдеров[cite: 1].
4. Удобная игротека: Позволяет добавлять новые игры через окно прямо в меню[cite: 1].
5. Безопасный выход: Автоматически возвращает рабочий стол в норму[cite: 1].

КАК ПОЛЬЗОВАТЬСЯ?
-------------------------------------------------------------------
1. Скачайте файлы "Games.bat" и "Игры.txt" и положите их в одну папку[cite: 1].
2. Запустите файл "Games.bat" обычным двойным кликом (права админа не нужны).
3. Рабочий стол скроется, откроется лаймовое меню терминала[cite: 1].
4. Выберите игру по цифре или нажмите "A", чтобы добавить новую[cite: 1].
5. При выборе пункта [0] ВЫХОД ваш рабочий стол полностью восстановится[cite: 1].

💡 СОВЕТ ДЛЯ УДОБСТВА И КРАСОТЫ:
Нажмите правой кнопкой мыши по файлу "Games.bat" -> Отправить -> Создать 
ярлык на рабочем столе. 
Вы можете нажать по созданному ярлыку правой кнопкой мыши -> Свойства -> 
Сменить значок -> Выбрать любую красивую иконку (например, в виде геймпада). 
Теперь запускать консоль можно красиво и стильно прямо с рабочего стола!

КАК НАСТРОИТЬ СВОЮ ПАПКУ ДЛЯ ВЫБОРА ИГР (В КОДЕ БАТНИКА)?
-------------------------------------------------------------------
По умолчанию при добавлении новой игры окно выбора файлов открывается 
на диске C:\[cite: 1]. Если все ваши игры хранятся в конкретном месте (например, 
в "D:\Games"), вы можете сделать так, чтобы окно сразу открывалось там.

Для этого:
1. Откройте файл "Games.bat" через Блокнот.
2. Найдите строку номер 96 (или через Ctrl+F найдите "InitialDirectory").
3. Замените параметр:
   $d.InitialDirectory='%SystemDrive%\';
   На путь к вашей папке. Например:
   $d.InitialDirectory='D:\Games';
4. Сохраните файл.

ЧТО ДЕЛАТЬ, ЕСЛИ СБОЙНУЛ РАБОЧИЙ СТОЛ?
-------------------------------------------------------------------
Если у вас выключили свет, зависла игра или вы случайно закрыли окно 
батника крестиком, из-за чего рабочий стол не вернулся сам[cite: 1]:
1. Нажмите Ctrl + Alt + Del и выберите "Диспетчер задач"[cite: 1].
2. Нажмите "Запустить новую задачу" (или Файл -> Запустить новую задачу)[cite: 1].
3. Введите explorer.exe и нажмите Enter. Рабочий стол вернется![cite: 1]
