# 🎮 Игровой лаунчер-оптимизатор для слабых ПК

Удобный интерактивный скрипт (`Games.bat`), который временно разгружает операционную систему, выгружая всё лишнее из оперативной памяти для повышения стабильности и FPS в играх[cite: 1, 2].

> 🤖 **Важное примечание об авторстве:** Весь этот проект — от самой идеи и логики батника до результатов тестов и личной истории — полностью придуман, разработан и протестирован автором. Нейросеть Gemini использовалась исключительно как текстовый инструмент, чтобы красиво оформить мысли автора и сделать описание лаконичным и читабельным.

---

## 💡 О проекте: Для каких игр это создано?

В первую очередь этот скрипт и вся сборка ориентированы на **одиночные игры**[cite: 1, 2]. Это практичный способ освободить ресурсы на тяжелых версиях Windows 11[cite: 1, 2].

### ➕ Плюсы использования:
* **Освобождение ресурсов:** Отключение графической оболочки Windows освобождает процессор и ОЗУ для запущенной игры[cite: 1, 2].
* **Чистота системы:** В фоне не будут запускаться случайные задачи, обновления или уведомления, которые могут вызывать фризы[cite: 1, 2].
* **Удобная игротека:** Новые игры можно добавлять прямо через интерфейс лаунчера с помощью графического окна выбора файлов[cite: 1, 2].
* **Безопасность:** Скрипт ничего не удаляет безвозвратно. При выходе система полностью возвращается в стандартный рабочий вид[cite: 1, 2].

### ➖ Минусы и жесткие ограничения:
* **Не используйте сторонние лаунчеры:** Батник создан **специально для пиратских версий игр** (или игр без DRM, запускаемых напрямую через собственный `.exe`)! Даже не пытайтесь запускать через него игры из Steam, Epic Games Launcher, EA App и других официальных клиентов[cite: 1, 2]. Они автоматически подтянут свои тяжелые фоновые сервисы, что полностью уничтожит весь смысл оптимизации[cite: 1, 2].
* **Скачивайте игры только с проверенных и надежных ресурсов.** Поскольку батник ориентирован на запуск без официальных лаунчеров, чистота самой игры критически важна. Если скачать релиз со скрытым майнером или вирусом, они заберут все ресурсы процессора, и никакая оптимизация скрипта уже не поможет.
* **Временное ограничение функций:** Пока вы находитесь в игре, у вас не будет доступа к рабочему столу, браузеру и мессенджерам (они принудительно закрываются ради максимальной экономии памяти)[cite: 1, 2].

## 🛠️ Как пользоваться

1. **Скачайте** файлы `Games.bat` и `Игры.txt` и положите их в одну папку[cite: 1, 2].
2. **Запустите** файл `Games.bat` (права администратора не требуются)[cite: 1, 2].
3. Рабочий стол скроется, и откроется текстовое меню терминала[cite: 1, 2].
4. Выберите игру по цифре или нажмите `A`, чтобы добавить новую[cite: 1, 2].
5. При выборе пункта `[0] ВЫХОД` ваш рабочий стол и все функции Windows полностью восстановятся[cite: 1, 2].

### 💡 Совет для удобства и красоты:
Нажмите правой кнопкой мыши по файлу `Games.bat` ➔ *Отправить* ➔ *Создать ярлык на рабочем столе*[cite: 1, 2]. 
В свойствах ярлыка нажмите *«Сменить значок»* и выберите любую красивую иконку (например, в виде геймпада)[cite: 1, 2]. Теперь запускать лаунчер можно прямо с рабочего стола[cite: 1, 2].

### ⚙️ Настройка стартовой папки для выбора игр:
По умолчанию окно добавления игр открывается на диске `C:\`[cite: 1, 2]. Если ваши игры хранятся, например, в `D:\Games`, можно сделать так, чтобы окно сразу открывалось там:
1. Откройте файл `games.bat` через Блокнот[cite: 1, 2].
2. Нажмите `Ctrl + F` и найдите строку: `selected_path.tmp`[cite: 1, 2].
3. Чуть правее в этой строке найдите параметр: `$d.InitialDirectory='%SystemDrive%\';`[cite: 1, 2].
4. Замените параметр на свой путь в двойных кавычках и без слэша на конце. Например: `$d.InitialDirectory="D:\Games";`
5. Сохраните файл (`Ctrl + S`)[cite: 1, 2].

📌 **Что делать, если сбойнул рабочий стол?**
Если у вас выключили свет, зависла игра или вы случайно закрыли окно батника на крестик, из-за чего рабочий стол не вернулся сам[cite: 1, 2]:
1. Нажмите `Ctrl + Alt + Del` и выберите **«Диспетчер задач»**[cite: 1, 2].
2. Нажмите *«Запустить новую задачу»* (или *Файл* ➔ *Запустить новую задачу*)[cite: 1, 2].
3. Введите `explorer.exe` и нажмите Enter. Рабочий стол вернется![cite: 1, 2]

## 🧠 Что конкретно делает батник?

1. **Временно останавливает Проводник Windows (`explorer.exe`):** Освобождает ОЗУ и разгружает процессор[cite: 1, 2].
2. **Закрывает фоновые программы:** Автоматически завершает работу Discord, Telegram, WhatsApp, Steam и Epic Games на время сессии, чтобы они не расходовали память в фоне[cite: 1, 2].
3. **Закрывает Google Chrome:** Принудительно завершает процессы браузера[cite: 1, 2].
4. **Чистит мусор:** Очищает временные папки системы (`Temp`), не затрагивая кэш шейдеров[cite: 1, 2].
5. **Восстановление системы:** При закрытии игрового сеанса автоматически перезапускает оболочку Windows[cite: 1, 2].

## 🛠️ Дополнительные утилиты для оптимизации

Для улучшения плавности игрового процесса можно использовать вместе с батником следующие программы[cite: 2]:

### 1. Настройка ISLC (Рекомендуется для 16 ГБ ОЗУ)
Программа *Intelligent Standby List Cleaner* очищает кэш памяти и снижает задержки, улучшая взаимодействие железа[cite: 1, 2].

**Параметры для настройки:**
* *The list size is at least* (Размер списка минимум): **1024 MB**[cite: 1, 2]
* *Free memory is less than* (Свободная память меньше): **2048 MB**[cite: 1, 2]
* Поставьте галочку: *«Enable custom timer resolution»* (Включить пользовательское разрешение таймера)[cite: 1, 2].
* *Wanted timer resolution* (Желаемое разрешение таймера): **0.5000**[cite: 1, 2]
* *Частота опроса ISLC (мс):* **1000**[cite: 1, 2]
* Нажмите кнопку **«Start»**[cite: 1, 2].

### 2. Настройка Special K (Для улучшения плавности и инпут-лага)
Утилита *Special K* помогает настроить правильный фреймпейсинг (время кадра) и обеспечивает хороший инпут-лаг[cite: 1, 2]. Вы можете выставить **любую частоту кадров на ваш вкус**, но утилита особенно полезна, если игра не дотягивает до стабильных 60 FPS и вам нужна высокая плавность на более низкой частоте кадра[cite: 2].

* **Шаг 1: Скачивание.** Рекомендуется использовать проверенную версию **v25.3.25** в виде ZIP-архива[cite: 1, 2].
* **Шаг 2: Локальная установка в игру (Внутренняя инъекция).**
  Крайне рекомендуется использовать именно локальную (внутреннюю) инъекцию, так как глобальный режим режим потребляет гораздо больше системных ресурсов[cite: 2].
  1. Скопируйте файл `SpecialK64.dll` из папки утилиты[cite: 1, 2].
  2. Вставьте его в корневую директорию игры, туда же, где лежит главный исполняемый (`.exe`) файл[cite: 1, 2]. 
  3. **Важно по переименованию:** Название `dxgi.dll` используется **строго для игр на DirectX 11, DirectX 12 и Vulkan**. Если вы запускаете игру на более старых версиях графического API (например, DirectX 9 или 10), посмотрите в интернете, как правильно переименовать файл для вашей версии, либо в крайнем случае используйте глобальную инъекцию[cite: 2].
* **Шаг 3: Настройка внутри игры.**
  Запустите игру через батник[cite: 1, 2]. Нажмите `Ctrl + Shift + Backspace` (или кнопку *Home* на геймпаде) для открытия меню Special K[cite: 1, 2].

#### 🕹️ Выбор режима плавности: Half V-Sync vs Latent Sync

В зависимости от вашего железа и задач, выберите один из двух способов синхронизации[cite: 2]:

* **Вариант А: Половинная вертикальная синхронизация (Half V-Sync)**
  *Оптимально, если вы хотите получить плавность уровня домашних консолей (например, PS4) без резких скачков.*[cite: 2]
  1. Перейдите в верхнюю вкладку **Display**[cite: 1, 2].
  2. Включите половинный всинк (**Half V-Sync**)[cite: 1, 2]. *Примечание: Если пункт не появляется сразу, укажите в этой же вкладке точную герцовку вашего монитора (например, 60 Hz), и выбор появится[cite: 1, 2].*
  3. **ВАЖНЫЙ ПОРЯДОК НАСТРОЙКИ:** Только после включения Half V-Sync перейдите на главный экран утилиты и включите ограничитель кадров (**Framerate Limit**) строго на значение 30 (или половину герцовки вашего экрана)[cite: 1, 2]. При мониторе 60 Гц это залочит игру на супер-стабильные 30 FPS с идеальным графиком времени кадра[cite: 1, 2].
  * **Плюсы:** Дает безупречный, ровный график фреймпейсинга (времени кадра)[cite: 2]. Картинка становится невероятно мягкой.
  * **Минусы:** Присутствует небольшой инпут-лаг (задержка ввода), свойственный обычной вертикальной синхронизации, но он ощущается гораздо приятнее и стабильнее, чем при стандартных методах Windows[cite: 2].

* **Вариант Б: Низкая задержка (Latent Sync)**
  *Альтернативный способ для тех, кому критически важен моментальный отклик управления.*[cite: 2]
  1. Вместо включения Half V-Sync перейдите в меню и активируйте функцию **Latent Sync**[cite: 2].
  * **Плюсы:** Обеспечивает минимальный, чистый инпут-лаг, сопоставимый с полной игрой без вертикальной синхронизации вообще[cite: 1, 2].
  * **Минусы (Жесткое требование):** Ваша видеокарта обязательно должна иметь запас мощности и не должна быть загружена более чем на 70%[cite: 1, 2]. Если запас есть — отклик будет идеальным[cite: 2]. Если видеокарта нагружена под 100% — по экрану пойдут горизонтальные полосы разрыва кадра (тиринг)[cite: 1, 2].

#### 🛠️ Дополнительные настройки в Special K:
1. **Процессор (Вкладка Advanced):** Включите использование спящих ядер (актуально для многопоточных CPU) и поменяйте приоритет процесса игры на *«High»* (Высокий)[cite: 1, 2].
2. **Для видеокарт Nvidia (Вкладка Advanced):** Включите функцию *Nvidia Reflex* в режиме `Reflex + Boost` внутри утилиты (в настройках самой игры Reflex в этом случае лучше выключить)[cite: 1, 2]. Если появились нестабильности — просто отключите её[cite: 1, 2].

## 📝 От автора: История создания и тильта

Этот проект, можно сказать, сделан «на коленке» исключительно на основе моих личных пожеланий, потребностей и видения идеального игрового процесса[cite: 2]. Он всё еще находится в статусе незавершенного, и мне предстоит многое доработать[cite: 2].

Для меня 30 кадров в секунду — это не просто цифра, это жесткий порог для игр[cite: 2]. На моем ПК практически ни одна нормальная современная игра не идет в стабильные 60 FPS[cite: 2]. Именно ради того, чтобы выжать максимум из 30 кадров, и затевался весь этот проект[cite: 2]. 

За этой кажущейся простотой скрывается **больше полугода непрерывных тестов, поисков и тяжелых экспериментов**[cite: 2]. Раньше я перепробовал кучу методов: пытался настроить лок кадров через RTSS (RivaTuner Statistics Server) вместе с вертикальной синхронизацией[cite: 2]. Но из-за этого намертво ломался показатель **1% Low FPS** (самое главное значение, отвечающее за микрофризы и общую плавность картинки)[cite: 2]. Я пробовал включать половинный всинк в драйвере Nvidia вместе с RTSS, но получал просто чудовищный, неиграбельный инпут-лаг[cite: 2]. Из-за этих постоянных дерганий, фризов и кисельного управления у меня было много тильта и настоящего геймерского выгорания — я долгое время вообще не хотел прикасаться к играм, потому что это не приносило никакого удовольствия[cite: 2].

Однако, пройдя через сотни проб и ошибок, я отбросил всё лишнее и нашел эту финальную формулу[cite: 2]. Связка точечной разгрузки системы через батник и глубокой настройки Special K вернула мне радость от прохождения игр даже на слабом железе[cite: 2]. Надеюсь, этот опыт сбережет ваши нервы и поможет вам![cite: 2]
